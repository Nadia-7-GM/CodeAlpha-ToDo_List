# TaskPro - CodeAlpha Internship Project

#Live Demo: 
TaskPro is a modern task management web application developed as part of the CodeAlpha Internship Program.

## 🚀 Features
- Add tasks with due date and reminder
- Ringtone alerts for task notifications
- Task categorization using color labels
- Add steps/subtasks inside each task
- File attachment support
- Dark and Light theme toggle
- Built-in Timer and Stopwatch

## 🎯 Purpose
This project helps users manage daily tasks efficiently with an interactive and user-friendly interface.

## 🛠️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
